Flex-Grow

item1
item2
item3
This property specifies what amount of space inside the flex container can take up if necessary. Always relative to the size of other items in the container.
flex-grow : 0;     // default value

.item1 {
     flex-grow: 1;
}
.item2 {
     flex-grow: 2;     // item 2 grows twice the size of item 1
}

}

Flex-Shrink

item1
item2
item3
Defines the ability of an item to shrink if necessary.
Relative to the other items in the container.
flex-shrink : 1;     //default value
Shrinking is possible only upto certain point. After which items will start to overflow.

.item1 {
     width: 150px;
     flex-shrink: 0;
}
.item2 {
     width: 150px;
}
.item3 {
     width: 150px;
}
// here only item2 and item3 will shrink because they have default value of flex-shrink (i.e 0) but item1 will have fixed width and wouldn't shrink when the window width is decreased or increased.

Flex

item1
item2
item3
Shorthand for flex-grow, flex-shrink and flex-basis.
flex : flex-grow flex-shrink flex-basis
flex: 0 1 auto;     //default value

.item1 {
     flex: 0 1 150px;
}

Variations :
/* One value, unitless number: flex-grow */
flex: 2;

/* One value, width/height: flex-basis */
flex: 10em;
flex: 30px;

/* flex: none | initial | auto */

/* Two values: flex-grow | flex-basis */
flex: 1 30px;

/* Two values: flex-grow | flex-shrink */
flex: 2 2;

Align-Self

item1
item2
item3
Used to control the alignment of individual flex items. Similar to align-items of flex container.
Default value is auto, i.e it inherits from parent align-items property.

.item1 {
     align-self: flex-start;
}
.item2 {
     align-self: flex-end;
}
.item3 {
     align-self: center;
}
